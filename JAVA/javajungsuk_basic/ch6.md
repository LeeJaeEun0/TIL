클래스는 제품의 설계도, 객체는 제품으로 속성(변수)와 기능(메서드)로 구성

클래스가 필요한 이유: 객체를 생성하려고

객체가 필요한 이유? 객체를 사용하려고

객체를 사용하는 이유는? 객체의 속성과 기능을 사용하려고
